# TASKPACK Step 4：C1 效果恢复与透明机制分级验证

## 1. 任务目标

在不改变已验证 C1 多视角交织链路的前提下，逐层恢复主项目的完整游戏画面效果。

当前路线不是继续做普通功能堆叠，而是优先把画面效果层级恢复到可交付状态，并按渲染机制风险分级让用户在 C1 实机上确认。

## 2. 当前已验证基线

- C1 参数：OpenstageAI 在线设备参数，设备 `ZX1393`。
- 投影：`three-converged`。
- 视图顺序：`reversed`。
- 视图图集：`8x5`，共 `40` 视图。
- 默认 parallax：`1.00`。
- 游戏相机：轻微 3/4 俯视角，动态变化只响应玩家操作和游戏事件。
- 普通屏 debug 窗口：dev 模式显示，用于 FPS、draw calls、triangles、对象数和运行态检查。
