# Visual QA Checklist（C1 3D 专用）

## A. 深度观感
- [ ] 玩家、敌机、地形三层在 C1 屏上可稳定分离
- [ ] 高频特效（弹幕、爆炸）不会引发明显深度跳变
- [ ] 快速滚屏时无大面积深度闪烁

## B. 可读性
- [ ] 敌弹在复杂背景中仍一眼可见
- [ ] Boss 弱点与危险前摇提示清晰
- [ ] 道具与关键交互物不会被 bloom 淹没

## C. 电影感与风格统一
- [ ] 高对比但不过曝（亮部有层次）
- [ ] 发光区域有主次，不是全屏同亮
- [ ] 各阶段风格统一但差异明确

## D. 性能稳定性
- [ ] 高强度战斗下帧时间无明显尖峰
- [ ] 粒子峰值可控，池化正常
- [ ] InstancedMesh / LOD 方案生效

## E. 回归检查
- [ ] 碰撞判定未受视觉改动影响
- [ ] HUD 与关键提示未被遮挡
- [ ] Boss 战流程与刷怪逻辑正常
