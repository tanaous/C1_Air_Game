- 说明：本次为代理验证，未连接实机 C1 显示器，实机最终确认需补测。

## 3. 场景状态检查

| 状态 | 观察结论 | 风险等级 |
|---|---|---|
| 静止 | 轮廓清晰，机鼻/主脊/翼面层级稳定 | 低 |
| 移动 | 倾斜动作期间轮廓未塌陷，主机体识别稳定 | 低 |
| 聚焦 | 座舱与导轨亮度抬升，主体仍可读 | 低 |
| 开火 | 喷口与导轨发光增强，未出现全机泛光 | 中低 |
| 自旋 | 灯光和材质联动明显，机体边缘可读 | 中低 |
| 受击 | 短时高亮反馈清晰，不遮挡主体结构 | 中低 |

## 4. C1 专项代理结论

- 中尺度体块优先，未引入高频碎线，符合交织输出稳定性要求。
- 发光集中在喷口、导轨、座舱边缘，避免了大面积 bloom 型污染。
- 使用受控假玻璃与 Fresnel 增强，避免了重透明折射路径的排序风险。

## 5. 回退点（必要时）

1. 关闭或下调 `FresnelRim` 强度（优先回退项）。
2. 将玩家机 `emissiveIntensity` 按状态整体下调 20%。
3. 将 hero rim light 强度回退至 1.45（常态）/ 2.1（自旋）。
4. 如果仍有串扰感，优先下调导轨发光而不是削弱主装甲材质。

## 6. 待实机补测项

1. C1 实机五状态巡检：静止、移动、聚焦、开火、自旋。
2. 暗背景/亮背景双场景下的轮廓可读性对比。
3. 长时间游玩（15 分钟以上）下是否出现视觉疲劳点或发光污染累积感。
